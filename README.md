# Spring Security Learning Project (Before JWT)

This repository contains a **Spring Boot + Spring Security** project built to understand the fundamentals of Spring Security **before introducing JWT-based authentication**.

The goal of this project is to build a strong foundation in:
- Authentication and Authorization
- Spring Security filter chain
- Securing endpoints
- Role-based access control
- How Spring Security works internally (without JWT for now)

---

## Features Implemented

- Spring Boot application with Spring Security
- Basic Authentication and/or Form Login
- Custom `SecurityFilterChain` configuration
- Role-based access control (RBAC)
- Secured and public endpoints
- Password encoding
- User authentication using:
    - In-memory users or
    - Database (if configured in your project)
- CSRF configuration (enabled/disabled as per learning needs)
